# robotics
the source code for all the robotics projects

Repository rules for coding agents live in `CODEX.md`.
Arduino Uno car hardware and behavior notes live in `arduino-uno-car/BEHAVIOR.md`.
Arduino Uno car setup commands live in `arduino-uno-car/SETUP.md`.
ESP32 bare 16-pin LCD wiring and setup live in `esp32-lcd/`.
ESP32 I2C backpack LCD wiring and setup live in `esp32-lcd-i2c/`.
Each project keeps its own `upload-sketch.sh` wrapper next to its sketch.
